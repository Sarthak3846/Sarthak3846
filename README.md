### 🛡️ S.H.I.E.L.D.
**Safety Degradation Trajectories in LLMs Under Multi-Turn Jailbreak Attacks**

Automated evaluation framework that moves past binary jailbreak metrics — tracks *how* safety erodes turn-by-turn using Time-to-Degradation, AUC, and Max Degradation Rate.

`Python` `FastAPI` `Gemini/Groq/Mistral APIs` `SciPy` `Matplotlib`

- Modular Input/Output Security Engines scoring response safety on a 0–5 scale
- Per-turn trajectory analytics with automated visual reporting
- Statistical comparison across models & attack categories
